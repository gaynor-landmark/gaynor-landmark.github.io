Colour Scheme Creator

A Collection of Technologies

This app can be used to assist the choice of a website colour scheme, allowing the user to try out their colours behind their selected images.

A colour-picker is used to fill the swatches of a blank scheme, which can then be applied as backgrounds or font colours. Alternatively, the swatches can be filled straight from the ColourLovers Inspiration palette. After logging-in with a GitHub account a user can name and save the schemes they have created and retrieve them again at a later date.

This project was created as a base for implementing and combining a number of technologies. It includes authorising with an account, a call to the ColourLovers.com website for inspiration, saving and retrieving from a database, and loading data to the screen using a compiled Handlebars template.
